# E-Commerce Sales & Customer Analytics (Internship Project)

## Project Overview
This project was completed as part of a certified Data Analytics internship. It focuses on analyzing e-commerce sales data to understand customer behavior, product performance, revenue trends, and regional demand.

## Tools & Technologies
- SQL
- Microsoft Excel
- Power BI
- Tableau

## Key KPIs
- Total Sales & Profit
- Order Volume
- Customer Segmentation
- Product Category Performance
- Regional Revenue Trends

## Analysis Workflow
- Cleaned and prepared raw datasets using Excel
- Performed SQL-based analysis on transactional data
- Built interactive dashboards in Power BI and Tableau
- Created a summary presentation for business stakeholders

## Business Insights
- Identified top-performing products and categories
- Analyzed repeat vs new customer contribution
- Detected seasonal sales trends
- Provided recommendations to improve sales performance

## Folder Structure
- data: Dataset files
- sql: SQL queries
- powerbi: Power BI dashboards
- tableau: Tableau dashboards
- presentation: Summary presentation
- certificate: Internship certificate
